## 설치

이번 포스팅에서는 테스팅 프레임워크인 Jest를 적용해서 유닛 테스트를 진행할 겁니다.        
기본적인 환경 설정을 위해 프로젝트 폴더를 하나 만들고 터미널을 열어 아래의 명령어를 입력해 줍니다. 

```shell
npm install --save-dev jest
or
yarn add --dev jest
```

```shell
{
  "scripts": {
    "test": "jest"
  }
}
```
